# Protein Design with Modern Generative AI

This repository is my learning workspace for exploring modern generative AI models in protein design.

The current project is an MDM2 binder design case study, but the repo is intended to grow into notes, experiments, and small workflows for other protein design problems as well.

Current topics:

- Understanding RFdiffusion Colab notebooks
- Designing de novo binders against target protein pockets
- Reading AlphaFold validation metrics such as pLDDT, PAE, RMSD, pTM, and ipTM
- Comparing how different generative protein design tools fit into a practical design workflow

Study notes are collected in [`study_notes/`](study_notes/), including a walkthrough for MDM2 binder design and how to interpret common RFdiffusion/AlphaFold metrics.

Experiment records and generated outputs are kept under [`rf/results/`](rf/results/). Project-level summaries and learning notes are written in `study_notes/` after a project reaches a useful checkpoint.
